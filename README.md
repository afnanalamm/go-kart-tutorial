# Go Kart Tutorial
Tutorial to making a version of the wooden petrol go-kart I made in Year 12, complete with materials, documentation, common errors, best practices, troubleshooting problems etc!

## What's included
- My Excel spreadsheet with all the parts I ended up using, estimated costs and Amazon links where possible.
- A SketchUP file with go-kart models from different iterations
- A written guide

## Parts required
### You might need some other parts based on what design you're going for

-	2x Rear , 2x Front wheels
-	x3 - 2"x4" 3m C24 timber
-	Honda GX200/GX160 Engine or a clone
-	Centrifugal Clutch
-	Rear gear - #420 chain size
-	420 chain
-	420 rear sprocket
-	1" bore #40/41 clutch
-	7/8" to 1" shaft adapter
-	KP003 pillow block bearing
-	M6 80mm x6 bolts for front axle
-	M8 bolts for engine
-	Castrol 1L 10W 30 engine oil
-	1" Bore Shaft Collar
-	4pcs T-Bolt hose clamp
-	Steel M10 180mm bolts x50
-	Hand gloves
-	Rear axle kit
-	Angle Brackets, M10 Washers
